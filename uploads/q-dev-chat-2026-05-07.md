## Rancangan Fitur Kehadiran

### Struktur Status Presensi
Hanya 3 status:
- **Hadir** — santri presensi dalam periode waktu
- **Izin** — otomatis dari data perizinan yang disetujui
- **Alpha** — tidak hadir & tidak ada izin (default jika periode habis & tidak presensi)

---

### Sistem Kartu Warna

**Kategori Jamaah** (21 kegiatan/minggu, Minggu Subuh — Sabtu Isya):

| Kartu | Kondisi Naik | Kondisi Turun |
|-------|-------------|---------------|
| Putih | Alpha < 7 | — |
| Kuning | Alpha ≥ 7 minggu ini | Alpha < 7 minggu berikutnya |
| Orange | Kuning + Alpha ≥ 7 minggu berikutnya | Alpha < 7 |
| Merah | Orange + Alpha ≥ 7 | Alpha < 7 |
| Hitam | Merah + Alpha ≥ 7 | Alpha < 7 |

**Kategori Ngaji** (14 kegiatan/minggu):

| Kartu | Kondisi |
|-------|---------|
| Putih | Alpha < 2 |
| Kuning | Alpha ≥ 2 |
| Merah | Kuning + Alpha ≥ 3 minggu berikutnya |

Naik/turun bertahap, tidak bisa loncat.
